树莓派4B安装Ubuntu20.04.1 LTS

发表于 2021-01-16 更新于 2021-01-17

闲来无事买了个树莓派4B4G回来，记录一下安装系统的过程和一些配置过程。

1.安装系统

首先去树莓派官网下载并安装Raspberry Pi Imager，用于烧录系统镜像。

之后去Ubuntu for RaspberryPi下载Ubuntu Server 20.04.1 LTS镜像。

打开安装好的Raspberry Pi Imager，选择下载的镜像和TF卡，开始烧录。

烧录完成后会自动推出TF卡，需要重新拔插，并编辑system-boot盘中的network-config文件，

找到如下段落，取消其注释，并替换<wifi network name>和<wifi password>为你的wifi账号密码。

之后拔出TF卡，插入树莓派的卡槽中，启动机器，等待机器初始化10分钟左右。

当路由器显示连接上树莓派后，再后台进行IP与MAC地址绑定，防止IP地址在reboot之后变动。

在路由器后台找到名为ubuntu的树莓派设备，记住它的ip地址，使用ssh连接到树莓派。

如果是用macOS的Terminal连接ssh，此处会有一个[yes/no/…]，填写yes并回车即可。

提示输入默认密码，默认密码为ubuntu，连接成功后会让你立即输入并重新输入新密码。

2.更改设备名、默认用户名和密码

更改ssh登陆设置。

把文件中的PermitRootLogin一项更改为yes。

设置root账户密码，输入以下指令后会提示输入root账户新密码，输入后再回车设置完成。

解锁root账户。

编辑hostname文件，将ubuntu改为你想要的设备名。

重启树莓派。

用root账户再次ssh连接树莓派，和第一次用ubuntu连接一样，不过输入的密码是刚才设置的root账户密码。
